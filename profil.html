Profil Aplikasi

Aplikasi ini dibuat untuk memenuhi kebutuhan penjualan barang secara online. Aplikasi ini dapat digunakan oleh pengguna untuk melakukan transaksi penjualan barang secara mudah dan cepat.

Aplikasi ini memiliki beberapa fitur, seperti:

  • Fitur transaksi penjualan barang
  • Fitur keranjang belanja
  • Fitur login

Aplikasi ini dibuat dengan menggunakan teknologi web seperti HTML, CSS, dan JavaScript. Aplikasi ini juga menggunakan database untuk menyimpan data transaksi dan pengguna.
